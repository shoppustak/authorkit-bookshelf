# AuthorKit Bookshelf

Public bookshelf for discovering books from indie authors powered by AuthorKit.

## URLs
- Production: https://bookshelf.authorkit.pro
- Vercel: https://authorkit-bookshelf.vercel.app

## Pages
- `/` - Homepage with featured books
- `/browse` - Browse all books with filters

## Development
This is a static HTML site deployed to Vercel.

## Deployment
Deployed automatically on push to main branch via Vercel.
